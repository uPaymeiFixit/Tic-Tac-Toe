Tic-Tac-Toe

Launch project
This is an infinitely expandable Tic-Tac-Toe game. The game also has an artificial intelligence option. I added a script to interact with an Arduino I have been experimenting with. When a player moves, it sends a command using PHP over the serial communication port to an Arduino, telling it the coordinates to apply ground and 5 volts. The Arduino uses my own method of multiplexing, and can only expand to a 5 by 5 LED grid.
